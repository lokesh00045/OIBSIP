# Temperature Converter Website

## Project Description

This is a responsive Temperature Converter Website developed as part of the **OIBSIP Level 1 - Task 3** internship.

The application allows users to convert temperatures between **Celsius, Fahrenheit, and Kelvin** instantly using JavaScript.

## Features

- Convert Celsius to Fahrenheit and Kelvin
- Convert Fahrenheit to Celsius and Kelvin
- Convert Kelvin to Celsius and Fahrenheit
- Input validation for empty or invalid values
- Responsive and user-friendly interface
- Clean and simple design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open the **LokeshBanoth_Task3** folder.
3. Open **index.html** in any modern web browser.
4. Enter a temperature value.
5. Select the input unit.
6. Click the **Convert** button to view the converted temperatures.
